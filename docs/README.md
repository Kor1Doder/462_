# Hardware documentation

Reference manuals and datasheets for the retrofitted machines and their
electronics. These are large binary documents kept for reference — they are not
needed to build or run the software.

## Machines

- **`hardware/emco/`** — EMCO F1-CNC Mill:
  - `EMCO Compact 5 & F1-CNC_ Service Manual.pdf`
  - `EMCO F1-CNC_ Basic.pdf`
  - `EMCO F1-CNC-P_ Service Parts.pdf`
  - `ME462 Rapor -- 31.3.26.pdf` — project report
  - `milling_hardware_and choices.docx` — hardware selection notes
- **`hardware/lmc/`** — Light Machines Corp. spectraLIGHT Lathe:
  - `LMC spectraLIGHT - CNC Lathe_ User's Guide.pdf`
- **`hardware/First_Look.docx`** — first-look overview of both machines.
